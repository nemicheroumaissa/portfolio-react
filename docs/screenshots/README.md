# README Screenshots

Add your portfolio screenshots here so they appear in the main `README.md`.

## Recommended files

| File | What to capture |
|------|-----------------|
| `preview.png` | Hero / full home view (used as main preview) |
| `home.png` | Home section |
| `about.png` | About / My Story |
| `skills.png` | Skills grid |
| `projects.png` | Projects section |
| `contact.png` | Contact section |
| `mobile.png` | Mobile layout (narrow viewport) |
| `case-study.png` | Project case study page (optional) |
| `dark-mode.png` | Dark theme (optional) |

## How to capture

1. Run `npm run dev` and open http://localhost:5173
2. Use your browser screenshot tool or **Win + Shift + S** (Windows)
3. Save images in this folder with the names above
4. Commit and push — they will display on GitHub automatically

**Tip:** Use [Screely](https://screely.com) or [shots.so](https://shots.so) to add a nice browser frame around your captures.
